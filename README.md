# Flappy Bird Game

## Overview
This project is a simple implementation of the classic Flappy Bird game. The player controls a bird that must navigate through gaps between pipes without colliding with them. The game keeps track of the player's score based on how many obstacles are successfully passed.

## Features
- Interactive Flappy Bird gameplay
- Score tracking system
- Collision detection with pipes and ground
- Simple and responsive controls

## Technologies Used
- Python
- Pygame

## How to Run
1. Install Python on your system.
2. Install the required library:
   pip install pygame
3. Run the main game file:
   python flappy_bird.py

## Controls
- Press **Spacebar** or **up arrow** to make the bird jump.
- Avoid hitting pipes or the ground.
